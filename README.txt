Installation
------------

Depending on your visualization system, please change to the corresponding
directory inside the 'modules' directory and follow the instructions there,
or simply do a 'make' (in case of UNIX/Linux).


Directories
-----------

doc                 : Documentation
libs                : Libraries (wrapping system dependent stuff)
modules             : The modules
sys                 : e.g. system dependent makefile components


Additional Information
----------------------

Please have a look at the "news" section in the doc/ directory or the corresponding web page of UniViz from time to time.
