Installation
------------

Depending on your visualization system, please change to the corresponding
directory and follow the instructions there, or simply do a 'make' (in case
of UNIX/Linux).


Directories
-----------
modules/avs         : modules for AVS 5.x
modules/covise      : modules for current Covise (>= 6)
modules/covise5     : modules for Covise 5.x
modules/paraview    : modules for current ParaView (>= 3.3, recommended)
modules/paraview2   : modules for ParaView 2.x
modules/paraview3.2 : modules for ParaView 3.0 to ParaView 3.2
